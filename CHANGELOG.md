# Changelog

## 0.1.0

- Initial publishable Sonos Now release.
- Textual command-line UI for Sonos discovery, metadata, album art, volume, playback, grouping, and debug status.
- Direct SoCo backend with no `soco-cli` subprocesses.
- Lazy Every Noise similar-artist suggestions.
- Color ASCII album art.
- Fullscreen Textual visualizer engines with hidden festival scene.
